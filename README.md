# gpa-app


## 取得単位記録アプリ

### 概要
大学での取得単位を記録・管理し、要件を満たしているかの判定を行うwebアプリ。卒業要件は、九州工業大学情報工学部知能情報工学科の必要単位に対応している。
### 作成した目的
大学で使用しているLiveCampus Uでは、単位の確認に少し時間がかかり、また、瞬時に取得単位・必要単位が理解しづらいため、取得単位、必要単位をすぐに確認できるようにするために作成した。

### 使用技術
・Python(Flask)
・SQLite
・HTML / CSS
・Git / GitHub

### 機能一覧
・取得単位の登録
・取得単位の一覧表示
・記録した単位の削除
・要件を満たしているかの表示

### 工夫した点
・作成するwebアプリケーションの内容を考え、軽量で簡単に扱えるSQLiteを使用した。SQLiteを用いてデータの永続化を実現した。
・入力される値が限られているところでは、プルダウンメニューにすることで操作性を向上させた。
・シンプルで直感的に使えるUIを意識した。

### 実行方法
1.リポジトリをクローン
2.以下のコマンドを実行
python app.py
